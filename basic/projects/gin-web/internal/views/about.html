About This Project

This is a production-ready Gin web application template for the Go Pro Learning Platform.

Technology Stack

  • Go 1.23+ - Modern Go with latest features
  • Gin v1.12 - High-performance HTTP web framework
  • HTML Templates - Go's html/template package
  • Docker - Multi-stage builds for small images

Project Structure

basic/projects/gin-web/
├── cmd/server/main.go        # Application entry point
├── internal/
│   ├── handler/              # HTTP handlers
│   ├── middleware/           # Custom middleware
│   └── views/                # HTML templates
├── static/                   # CSS and JS assets
├── Dockerfile
└── docker-compose.yml
