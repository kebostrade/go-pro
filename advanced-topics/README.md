## 🎯 Learning Path

### Prerequisites
- Go 1.23+ installed
- Basic Go knowledge (variables, functions, structs)
- Familiarity with command line
- Docker installed (for microservices topics)

### Recommended Flow
1. **Start Here**: System Design → REST APIs → CLI Apps
2. **Web Development**: Testing → Gin → WebSockets
3. **Distributed Systems**: Microservices → gRPC → Event-Driven
4. **Cloud Native**: Kubernetes → Serverless
5. **Specialized**: ML → Blockchain → IoT → GraphQL

## 🛠 Quick Start

```bash
# Navigate to any topic
cd advanced-topics/[topic-number]

# Run examples
go run main.go

# Run tests
go test -v ./...

# Build and run (for some topics)
go build -o app && ./app
```
