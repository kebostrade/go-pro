### Step 3: Write Your Script (15 min)

**60-Second Script Template**:

```
[0-5 seconds] HOOK
"Stop scrolling! I just discovered [something amazing]..."

[5-15 seconds] PROMISE
"In the next 60 seconds, I'll show you how to [achieve result]..."

[15-45 seconds] DELIVER
"Here's how it works:
1. [First step]
2. [Second step]
3. [Third step]"

[45-60 seconds] CALL TO ACTION
"Try this today and let me know your results! 
Follow for more [niche] tips."
```

**Example Script** (AI Tools Niche):
```
[HOOK - 0-5s]
"I made $1,000 in 24 hours using AI. Here's how."

[PROMISE - 5-15s]
"I'll show you the exact 3 AI tools I used and 
how you can start today, completely free."

[DELIVER - 15-45s]
"Tool 1: ChatGPT - I used it to write 10 viral scripts
Tool 2: Veo 3 - Generated all my video content
Tool 3: ElevenLabs - Created professional voiceovers
Total cost? $0. Time invested? 2 hours."

[CTA - 45-60s]
"Want the full tutorial? Follow me and comment 'AI' 
for the free guide. Let's build together!"
```

**Word Count Guide**:
```
5 seconds = 10-15 words
15 seconds = 30-40 words
30 seconds = 60-75 words
60 seconds = 120-150 words
```

### Step 6: Edit Video (15 min)

**Using CapCut** (Desktop or Web):

**Import**:
```
1. Open CapCut
2. Create new project (9:16 for vertical)
3. Import all video clips
4. Import voiceover MP3
```

**Arrange Timeline**:
```
1. Drag voiceover to audio track
2. Align video clips with voiceover:
   - Clip 1: 0-5s (Hook)
   - Clip 2: 5-45s (Main)
   - Clip 3: 45-60s (CTA)
3. Trim clips to match audio
```

**Add Transitions**:
```
1. Click between clips
2. Add "Dissolve" or "Zoom" transition
3. Duration: 0.5 seconds
```

**Add Text Overlays**:
```
1. Click "Text"
2. Add key phrases:
   - "$1,000 in 24 hours"
   - "3 Free AI Tools"
   - "Follow for more"
3. Style: Bold, large, high contrast
4. Animation: Fade in/out
```

---

### Step 7: Add Music & Effects (15 min)

**Background Music**:
```
1. CapCut → Audio → Sounds
2. Search: "upbeat" or "motivational"
3. Add to timeline
4. Lower volume to 20-30%
5. Fade in/out at start/end
```

**Sound Effects** (Optional):
```
- "Whoosh" for transitions
- "Ding" for key points
- "Pop" for text animations
```

**Captions** (Highly Recommended):
```
1. CapCut → Text → Auto Captions
2. Select language
3. Generate
4. Review and fix errors
5. Style: Bold, white text, black outline
```

**Color Grading**:
```
1. Select all clips
2. Adjustments → Brightness: +10
3. Saturation: +15
4. Contrast: +10
5. Makes video pop!
```
