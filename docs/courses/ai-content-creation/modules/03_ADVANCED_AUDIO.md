## 🎚️ Professional Audio Mixing

### The Perfect Mix Formula

**Volume Levels** (Relative to dialogue at 0 dB):
```
Dialogue/Narration:     0 dB (100%)
Music (Background):    -12 dB (25%)
Sound Effects:         -6 dB (50%)
Ambient Noise:        -18 dB (12%)
```

**Frequency Balance**:
```
Dialogue:    200 Hz - 3 kHz (clarity)
Music:       60 Hz - 15 kHz (fullness)
SFX:         100 Hz - 8 kHz (impact)
Ambient:     40 Hz - 10 kHz (atmosphere)
```

### Audio Ducking Technique

**What is Ducking?**
Automatically lower background audio when dialogue plays.

**CapCut Ducking Setup**:
```
1. Select background music track
2. Click "Audio" → "Ducking"
3. Set intensity: 60-80%
4. Set fade time: 0.3-0.5 seconds
5. Apply to all dialogue sections

Result: Crystal clear dialogue, professional sound
```

**Manual Ducking** (Advanced):
```
When dialogue starts:
- Lower music to -18 dB (0.3s fade)
- Lower ambient to -24 dB (0.3s fade)
- Keep SFX at normal level

When dialogue ends:
- Raise music to -12 dB (0.5s fade)
- Raise ambient to -18 dB (0.5s fade)
```
