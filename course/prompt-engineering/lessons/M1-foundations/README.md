# Module 1: Foundations

Learn the fundamentals of prompt engineering and how LLMs work.

## Lessons

| Lesson | Topic | Duration |
|--------|-------|----------|
| [01](01-introduction/README.md) | Introduction to Prompt Engineering | 2h |
| [02](02-llm-basics/README.md) | How LLMs Work: Tokens, Context, Attention | 3h |
| [03](03-prompt-anatomy/README.md) | Anatomy of a Good Prompt | 2h |
| [04](04-zero-shot/README.md) | Zero-Shot Prompting | 2h |

## Learning Objectives

- Understand what prompt engineering is and why it matters
- Learn how LLMs process and respond to prompts
- Master the basic structure of effective prompts
- Apply zero-shot prompting techniques

## Prerequisites

- Basic programming knowledge
- Familiarity with AI/LLMs helpful but not required

## Estimated Completion Time

Week 1-2 (9 hours total)
