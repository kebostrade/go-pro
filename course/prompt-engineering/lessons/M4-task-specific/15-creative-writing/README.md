# PE-15: Creative Writing & Content

**Duration**: 2 hours
**Module**: 4 - Task-Specific Prompting

## Learning Objectives

- Generate various content types (blogs, emails, stories)
- Control tone, style, and voice
- Adapt content for different audiences
- Edit and refine AI-generated content

## Content Types

### Blog Posts

```
Write a blog post about [topic].

SPECIFICATIONS:
- Length: 800-1000 words
- Tone: Professional but conversational
- Audience: [specific audience]
- Goal: [inform/persuade/entertain]

STRUCTURE:
1. Hook (engaging opening)
2. Problem statement
3. Main content (3-5 sections with headers)
4. Practical examples/tips
5. Conclusion with CTA

STYLE:
- Short paragraphs (2-3 sentences)
- Use bullet points for lists
- Include relevant statistics
- Add subheadings every 200 words

SEO:
- Primary keyword: [keyword]
- Include in title, first paragraph, and 2-3 times naturally
- Meta description: 155 characters

AVOID:
- Passive voice
- Jargon without explanation
- Walls of text
```

### Email Templates

```
Create an email template for [purpose].

EMAIL TYPE: [welcome/announcement/follow-up/sales]

STRUCTURE:
- Subject line: [under 50 chars, compelling]
- Preview text: [appears in inbox preview]
- Greeting: [appropriate to relationship]
- Body: [main content]
- CTA: [single, clear action]
- Sign-off: [appropriate closing]

TONE: [formal/casual/warm/professional]

PERSONALIZATION TOKENS:
- {{first_name}}
- {{company_name}}
- {{last_action}}

VARIATIONS:
Create 3 subject line options with A/B testing potential.

EXAMPLE DATA:
- Recipient: Marketing Manager at tech company
- Context: Following up after demo call
- Goal: Schedule next meeting
```

### Social Media Posts

```
Create social media posts for [campaign/topic].

PLATFORMS:
- LinkedIn: Professional tone, 300 words max
- Twitter/X: 280 chars, 1-2 hashtags
- Instagram: Engaging caption, 5-10 hashtags

POST REQUIREMENTS:
- Hook in first line
- Value proposition clear
- CTA included
- Platform-appropriate length

VARIATIONS:
Create 3 versions for each platform:
1. Educational
2. Engagement-focused (question/poll)
3. Promotional

HASHTAG STRATEGY:
- 2-3 broad hashtags
- 2-3 niche hashtags
- 1 branded hashtag

POST SCHEDULING CONTEXT:
- Industry: [industry]
- Best posting times: [times]
- Target audience: [audience]
```

### Marketing Copy

```
Write marketing copy for [product/service].

PRODUCT:
- Name: [product name]
- Category: [category]
- Price: [price]
- Key features: [3-5 features]

DELIVERABLES:
1. Headline (10 words max, compelling)
2. Subheadline (20 words, expands headline)
3. Feature bullets (3-5 bullets)
4. Social proof snippet (testimonial format)
5. CTA button text (3-5 words)

COPY FRAMEWORK: AIDA
- Attention: Hook the reader
- Interest: Build curiosity
- Desire: Create want
- Action: Clear next step

TONE: [premium/friendly/urgent/helpful]

UNIQUE SELLING PROPOSITION:
[What makes this different from competitors]

TARGET AUDIENCE:
[Demographics, pain points, desires]
```

## Voice & Tone Control

### Tone Spectrum

```
Write [content] with these tone specifications:

TONE POSITION: [1-10 scale]
1 = Very Formal → 10 = Very Casual
Current target: [X]

FORMAL EXAMPLES:
- "We regret to inform you..."
- "Please be advised that..."

CASUAL EXAMPLES:
- "Hey, just wanted to let you know..."
- "Heads up - here's the deal..."

PERSONALITY TRAITS:
- Warmth: [1-10]
- Humor: [1-10]
- Authority: [1-10]
- Enthusiasm: [1-10]

BRAND VOICE:
[Describe brand personality in 3 adjectives]
```

### Style Transfer

```
Rewrite this content in the style of [persona/publication]:

ORIGINAL:
[content to rewrite]

TARGET STYLE:
- Vocabulary level: [academic/accessible/technical]
- Sentence structure: [complex/varied/simple]
- References: [pop culture/industry/classic]
- Signature elements: [what makes this style unique]

EXAMPLES OF TARGET STYLE:
[1-2 examples of the target style]

PRESERVE:
- Core message
- Key facts
- Call to action
```

## Creative Writing

### Storytelling

```
Write a short story with these parameters:

GENRE: [genre]
LENGTH: [word count]
POV: [first/third person]
TENSE: [past/present]

STORY ELEMENTS:
- Protagonist: [description]
- Setting: [time and place]
- Conflict: [internal/external]
- Theme: [central theme]

STRUCTURE:
1. Opening hook (first paragraph)
2. Inciting incident
3. Rising action (2-3 complications)
4. Climax
5. Resolution

STYLE NOTES:
- Show, don't tell
- Use sensory details
- Include dialogue
- Vary sentence length

AVOID:
- Clichés
- Deus ex machina endings
- Overly convenient coincidences
```

### Character Development

```
Create a detailed character profile:

BASIC INFO:
- Name: [full name]
- Age: [age]
- Occupation: [job]
- Location: [where they live]

PERSONALITY:
- MBTI: [type] (or describe)
- Key traits: [3-5 adjectives]
- Flaws: [2-3 weaknesses]
- Strengths: [2-3 strengths]

BACKGROUND:
- Family: [brief family situation]
- Education: [level and field]
- Defining moment: [shaped who they are]

PSYCHOLOGY:
- Fears: [what keeps them up at night]
- Desires: [what they want most]
- Secrets: [what they hide]
- Contradictions: [what makes them complex]

BEHAVIOR:
- Speech patterns: [how they talk]
- Body language: [typical gestures]
- Habits: [daily rituals]
- Quirks: [unique behaviors]

RELATIONSHIPS:
- Friends: [who they're close to]
- Enemies: [who they conflict with]
- Romantic: [relationship status]
```

### Dialogue Writing

```
Write dialogue for this scene:

CHARACTERS:
- Alice: [brief description]
- Bob: [brief description]

CONTEXT:
- Setting: [where]
- Situation: [what's happening]
- Tension level: [1-10]

DIALOGUE GOALS:
- Alice wants: [her goal]
- Bob wants: [his goal]
- Subtext: [what's unsaid]

FORMAT:
- Include action beats
- Show emotion through behavior
- Use distinct voices for each character

RULES:
- No perfect responses (interruptions, overlaps)
- Natural speech patterns (um, pauses)
- Subtext over exposition
- Advance plot or character with each line
```

## Editing & Refinement

### Content Editing

```
Edit this content for improvement:

CONTENT:
[original content]

EDITING FOCUS:
1. Clarity: Remove ambiguity
2. Conciseness: Cut unnecessary words
3. Flow: Improve transitions
4. Engagement: Strengthen hooks
5. Accuracy: Verify facts

OUTPUT:
- Edited version
- Change log (what was changed and why)
- Suggestions for further improvement

TRACK CHANGES:
- Strikethrough for deletions
- Bold for additions
- [Comment] for suggestions
```

### Style Polish

```
Polish this content to match our style guide:

STYLE GUIDE:
- Voice: [description]
- Tone: [specifications]
- Vocabulary: [level and preferences]
- Sentence length: [target average]
- Paragraph length: [target]

CHECKLIST:
□ Active voice preferred
□ No jargon without context
□ Parallel structure in lists
□ Consistent tense
□ Varied sentence openings
□ Strong verbs over adverbs

ORIGINAL:
[content]

POLISHED:
[edited content]

CHANGES MADE:
[List key improvements]
```

1. Welcome + getting started
2. Feature highlight
3. Engagement check-in

### Exercise 15.3: Style Adaptation

- Social media (casual, engaging)
- Internal memo (direct, informative)
- Customer email (warm, helpful)

## Key Takeaways

- ✅ Specify content type, length, tone, and audience
- ✅ Use frameworks (AIDA, story structure) for guidance
- ✅ Define voice through examples and parameters
- ✅ Request variations for testing
- ✅ Edit AI output with specific improvement goals
