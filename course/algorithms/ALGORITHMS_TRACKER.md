## 📝 Pattern Recognition Notes

### Sliding Window

**When to use**: Contiguous subarrays/substrings, "longest/shortest" with constraint

**Template**:
```go
left := 0
for right := 0; right < len(arr); right++ {
    // expand window
    for !valid(window) {
        // shrink from left
        left++
    }
    // update result
}
```

### Two Pointers

**When to use**: Sorted arrays, palindromes, pairs with sum

**Template**:
```go
left, right := 0, len(arr)-1
for left < right {
    sum := arr[left] + arr[right]
    if sum == target { return }
    if sum < target { left++ } else { right-- }
}
```

### Fast & Slow Pointers

**When to use**: Cycle detection, find middle

**Template**:
```go
slow, fast := head, head
for fast != nil && fast.Next != nil {
    slow = slow.Next
    fast = fast.Next.Next
    if slow == fast { return true } // cycle
}
```

### BFS (Level Order)

**When to use**: Shortest path, level-by-level processing

**Template**:
```go
queue := []*Node{root}
for len(queue) > 0 {
    levelSize := len(queue)
    for i := 0; i < levelSize; i++ {
        node := queue[0]
        queue = queue[1:]
        // process node
        for _, child := range node.Children {
            queue = append(queue, child)
        }
    }
}
```

### DFS (Recursive)

**When to use**: Exhaustive search, backtracking

**Template**:
```go
func dfs(node *Node, state State) {
    if baseCase { return }
    for _, neighbor := range neighbors {
        if !visited[neighbor] {
            visited[neighbor] = true
            dfs(neighbor, newState)
        }
    }
}
```

### Binary Search

**When to use**: Sorted data, find specific value

**Template**:
```go
left, right := 0, len(arr)-1
for left <= right {
    mid := left + (right-left)/2
    if arr[mid] == target { return mid }
    if arr[mid] < target { left = mid + 1 } else { right = mid - 1 }
}
```
