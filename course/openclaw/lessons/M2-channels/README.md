## Learning Outcomes

- Create and configure bots for major messaging platforms
- Understand channel access control
- Manage multi-channel deployments

## Supported Channels

OpenClaw supports numerous channels:

| Channel | Difficulty | Notes |
|---------|-----------|-------|
| Telegram | ⭐ Easy | Best for beginners |
| Discord | ⭐ Easy | Great for communities |
| WhatsApp | ⭐⭐ Medium | Requires QR pairing |
| Slack | ⭐⭐ Medium | Enterprise features |
| Signal | ⭐⭐ Medium | Privacy-focused |
| iMessage | ⭐⭐⭐ Hard | Mac required |
