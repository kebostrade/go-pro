# Module 1: Getting Started

This module covers OpenClaw fundamentals, installation, and initial configuration.

## Lessons

| # | Lesson | Duration |
|---|--------|----------|
| 1 | [What is OpenClaw? Architecture Deep Dive](01-introduction/README.md) | 2h |
| 2 | [Installation: Homebrew, Docker, Source](02-installation/README.md) | 2h |
| 3 | [LLM Providers & Initial Configuration](03-configuration/README.md) | 2h |

## Learning Outcomes

- Understand the three pillars of OpenClaw: Gateway, Nodes, Skills
- Install OpenClaw using your preferred method
- Configure at least one LLM provider
- Verify installation with `openclaw doctor`

## Prerequisites

Before starting this module:
- ✅ Have an API key ready (OpenAI or Anthropic)
- ✅ Node.js 18+ installed (or Docker for container approach)
- ✅ Basic command line proficiency
