# OpenClaw: Build Your Own AI Agent

A comprehensive, hands-on course on building and deploying self-hosted AI agents with OpenClaw—the fastest-growing open-source AI agent platform with 150K+ GitHub stars.

## Course Overview

**Duration**: 4 weeks (self-paced)  
**Level**: Beginner to Intermediate  
**Prerequisites**: Basic command line knowledge, API keys (OpenAI/Anthropic), Node.js 18+

## Learning Objectives

By the end of this course, you will be able to:
- Understand OpenClaw architecture (Gateway, Nodes, Skills)
- Install and configure OpenClaw on any platform
- Connect multiple messaging channels (Telegram, Discord, WhatsApp)
- Extend OpenClaw with Skills and custom tools
- Deploy production-ready OpenClaw instances
- Secure AI agents against common vulnerabilities
