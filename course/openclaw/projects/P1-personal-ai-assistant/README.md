# Project 1: Personal AI Assistant

**Difficulty**: Beginner  
**Duration**: 3-4 hours  
**Prerequisites**: Lessons 1-3

## Project Overview

Build a personal AI assistant that you can message via Telegram to get help with daily tasks.

## Requirements

1. **Install OpenClaw** using Docker
2. **Configure Anthropic** or OpenAI provider
3. **Connect Telegram** bot
4. **Test conversations**:
   - Ask questions
   - Get reminders
   - Simple calculations

## Deliverables

- Working Telegram bot
- Screenshot of conversation
- Configuration files (without API keys)

## Extension Ideas

- Add memory skill for conversation history
- Enable web search skill
