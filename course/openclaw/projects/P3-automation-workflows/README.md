# Project 3: Custom Skills & Automation

**Difficulty**: Intermediate  
**Duration**: 5-6 hours  
**Prerequisites**: Lessons 7-8

## Project Overview

Create custom Skills to automate specific tasks.

## Choose One

### Option A: Weather Skill
- Fetch weather data from API
- Support multiple cities
- Cache results

### Option B: RSS Reader Skill
- Parse RSS/Atom feeds
- Summarize articles
- Scheduled updates

### Option C: Task Manager Skill
- Create/update/delete tasks
- Persist to file/DB
- Due date reminders

## Requirements

1. Create Skill manifest
2. Implement at least 2 tools
3. Install and test
4. Document usage

## Deliverables

- Working custom Skill
- Source code
- Usage examples

## Extension

- Publish to community
- Add tests
- Support configuration
