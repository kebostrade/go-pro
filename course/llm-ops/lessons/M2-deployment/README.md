# Module 2: LLM Deployment & Serving

Learn how to deploy and serve LLMs at scale using modern inference servers, API gateways, containerization, and Kubernetes orchestration.

## Lessons

| Lesson | Topic | Duration |
|--------|-------|----------|
| [IO-05](05-model-serving/README.md) | Model Serving Architectures | 3h |
| [IO-06](06-api-gateway/README.md) | API Gateway Patterns | 2h |
| [IO-07](07-containerization/README.md) | Containerization with Docker | 3h |
| [IO-08](08-kubernetes/README.md) | Kubernetes for AI | 3h |

## Learning Objectives

- Understand model serving architectures and inference optimization
- Implement API gateway patterns for LLM traffic management
- Containerize LLM applications for consistent deployment
- Deploy LLM services on Kubernetes with GPU support

## Prerequisites

- Completion of Module 1 (Foundations of AI Infrastructure)
- Basic understanding of REST APIs and HTTP protocols
- Familiarity with Docker containers
- Go programming experience (for code examples)

## Estimated Completion Time

Weeks 3-4 (11 hours total)

## Key Themes

1. **Inference Optimization**: Maximize throughput and minimize latency for LLM serving
2. **API Design**: Build robust, scalable APIs for LLM interactions
3. **Containerization**: Package and deploy LLM applications reproducibly
4. **Orchestration**: Scale LLM services with Kubernetes
