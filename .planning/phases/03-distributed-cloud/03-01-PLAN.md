  <files>basic/projects/kubernetes/api/v1alpha1/groupversion_info.go,basic/projects/kubernetes/api/v1alpha1/gopro_types.go,basic/projects/kubernetes/api/v1alpha1/zz_generated.deepcopy.go,basic/projects/kubernetes/controllers/suite_test.go,basic/projects/kubernetes/controllers/gopro_controller.go,basic/projects/kubernetes/cmd/main.go</files>
  <action>
    Create Kubernetes operator scaffold using kubebuilder/controller-runtime:

    1. **api/v1alpha1/groupversion_info.go** - Group version configuration:
       ```go
       package v1alpha1

       import (
           "k8s.io/apimachinery/pkg/runtime/schema"
           "sigs.k8s.io/controller-runtime/pkg/scheme"
       )

       var (
           GroupVersion = schema.GroupVersion{Group: "gopro.example.com", Version: "v1alpha1"}
           SchemeBuilder = &scheme.Builder{GroupVersion: GroupVersion}
           AddToScheme = SchemeBuilder.AddToScheme
       )
       ```

    2. **api/v1alpha1/gopro_types.go** - CRD type definitions:
       ```go
       package v1alpha1

       type GoProSpec struct {
           Image string `json:"image,omitempty"`
           Replicas *int32 `json:"replicas,omitempty"`
           Env map[string]string `json:"env,omitempty"`
       }

       type GoProStatus struct {
           AvailableReplicas int32 `json:"availableReplicas,omitempty"`
       }

       type GoPro struct {
           metav1.TypeMeta   `json:",inline"`
           metav1.ObjectMeta `json:"metadata,omitempty"`
           Spec   GoProSpec   `json:"spec,omitempty"`
           Status GoProStatus `json:"status,omitempty"`
       }
       ```

    3. **controllers/gopro_controller.go** - Reconcile loop:
       - Implement Reconciler interface from controller-runtime
       - Reconcile(ctx, req) returns Result, error
       - Fetch GoPro CR by name
       - Create/update Deployment based on Spec
       - Update Status with AvailableReplicas
       - SetupWithManager(mgr)

    4. **cmd/main.go** - Operator entry point:
       - Create manager with kubeconfig or in-cluster config
       - Register Gopro type with scheme
       - Register controller with reconciler
       - Start manager

    5. **controllers/suite_test.go** - Controller tests:
       - envtest.Environment setup
       - Test GoproReconciler

    Per kubebuilder pattern: Use controller-runtime for operator logic.
