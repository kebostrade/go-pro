# gqlgen Schema-First Workflow
```
1. Define schema in schema.graphqls
2. Run: gqlgen generate
3. Implement resolver interfaces
4. Wire up server
```

# GraphQL Schema Pattern (from existing reference)
```graphql
scalar Time

type User {
  id: ID!
  username: String!
  email: String!
  role: Role!
  posts: [Post!]!
  createdAt: Time!
}

type Query {
  user(id: ID!): User
  users: [User!]!
  me: User!
}

type Mutation {
  createUser(input: CreateUserInput!): User!
  deleteUser(id: ID!): Boolean!
}

type Subscription {
  userCreated: User!
}
```
